HTML

структура сайта, язык гипертекстовой разметки, HyperText Markup Language (эйчтиэмл)

CSS

стили сайта - Cascading Style Sheets (сиэсэс)

Основные теги

img (аймж)

тэг для изображения, атрибуты: src, alt alt - описание картинки src - путь к картинке, передаем через точку и слэш (./). Src (срси) – путь, Alt – описание картинки

h1-h2-h3-h4-h5-h6

размеры текста (h1 – большой, h6 – маленький)

p

параграф, абзац, нежирный по умолчанию

a

гипперссылка, атрибуты:href href - для ссылок

hr

создаёт линию на сайте (ашэр)

button

кнопка

Link

связывает документы, html c css

Тэги

Элементы в html в угловых скобках

Атрибуты

вспомогательная информация о тэге, которая определяет как будет выглядеть и работать =” ”

Input

поле ввода для данных, есть разные типы: text, number, email, color, radio, password и тд.

Метаданные (meta)

данные о сайте, находятся в head и не видны пользователю

display:flex

из блочных элементов делает строчные.
блочные элементы – занимают всю ширину сайта строчные элементы – занимают только свою ширину

justify-content

выравнивает элементы по главной оси (работает только когда есть display:flex)

align-items

выравнивает элементы по поперечной оси (работает только когда есть display:flex)

gap

расстояние между элементами работает только когда есть display:flex
